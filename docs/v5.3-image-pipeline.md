# v5.3｜Image Pipeline

## 新增

```text
studio/image_pipeline.py
studio/og_generator.py
content/assets/covers/
content/assets/heroes/
content/assets/illustrations/
assets/og/
assets/thumbs/
docs/v5.3-image-pipeline.md
```

## 功能

- 掃描圖片素材
- 建立 `data/image-manifest.json`
- 比對文章 slug 與封面圖
- 檢查缺少封面的文章
- 產生預設 OG SVG 圖

## 使用

```bash
python studio/image_pipeline.py
python studio/og_generator.py
```

## 建議命名

文章：

```text
content/stories/2026-07-02-dodo-swimming.md
```

封面：

```text
content/assets/covers/dodo-swimming.png
```

這樣系統能自動依 slug 對應。
