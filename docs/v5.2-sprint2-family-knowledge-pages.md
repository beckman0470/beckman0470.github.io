```text
studio/family_pages.py

```

## 功能

依照 `data/stories.json` 中每篇文章的 `characters` 欄位，自動產生人物頁。

每頁包含：

- 人物介紹
- 相關標籤
- 相關系列
- 出現文章
- 人物時間軸

## 使用

```bash
python studio/family_pages.py
```

## 建議整合到 build.py

```python
from studio.family_pages import build_family_pages
build_family_pages(ROOT)
```
