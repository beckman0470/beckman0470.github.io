# v3.5 Navigation & Subscription Integration

## 新增

- `subscribe.html`
- `docs/v3.5-navigation-subscription.md`
- `tools_update_nav.py`（選用）

## 這版目的

把 v3.2～v3.4 完成的功能正式接進網站動線：

- 搜尋 `search.html`
- 標籤 `tags.html`
- 訂閱 `subscribe.html`
- RSS `rss.xml`
- Sitemap `sitemap.xml`

## 建議上傳方式

先上傳：

```text
subscribe.html
docs/v3.5-navigation-subscription.md
tools_update_nav.py
```

如果你目前不方便執行 Python，沒關係，至少 `subscribe.html` 可以先獨立使用。
