# v5.8 Sprint 3｜Related Stories Block

## 交付檔案

```text
templates/components/related-stories.html
templates/components/related-story-card.html
templates/components/related-stories-example.html
css/related-stories.css
docs/v5.8-sprint3-related-stories-block.md
```

## 用途

建立文章頁底部「你可能也會喜歡」區塊。

## 原則

- 先做成可手動插入的 HTML 模組。
- 不自動改正文。
- 不自動推薦文章。
- 後續 Sprint 才接自動 Related Stories 邏輯。

## 使用方式

在文章頁 `<head>` 加入：

```html
<link rel="stylesheet" href="../css/related-stories.css">
```

在文章正文後插入：

```html
<!-- templates/components/related-stories-example.html -->
```
