# v3.1 Auto Article Generator

## 這版完成什麼

從：

```text
content/stories/*.md
```

自動產生：

```text
data/stories.json
articles/{slug}.html
sitemap.xml
```

## 使用方式

```bash
python cms/build.py
```

## 支援 Markdown

- `#`
- `##`
- `###`
- 一般段落
- `> 引言`

## 新增文章流程

1. 複製 `templates/story-template.md`
2. 放到 `content/stories/YYYY-MM-DD-slug.md`
3. 修改 front matter
4. 寫正文
5. 執行 `python cms/build.py`
6. Commit 產出的檔案
