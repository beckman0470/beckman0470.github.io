# v3.2 Full Site Search

這版新增 `search.html`，讓網站可以透過 `data/stories.json` 搜尋內容。

## 搜尋範圍

每篇故事的：

- title
- summary
- series
- category
- tags
- characters
- research

## 使用方式

把以下檔案放到 Repository：

```text
search.html
js/search-page.js
docs/v3.2-full-site-search.md
```

然後開啟：

```text
https://beckman0470.github.io/search.html
```
