# v5.4 Sprint 2.1｜Typography System

## 新增

```text
assets/css/typography.css
tools_article_typography_patch.py
docs/v5.4-sprint2-1-typography-system.md
```

## 功能

建立文章閱讀排版系統：

- 最佳閱讀寬度 72ch
- 字級階層
- 行距
- Pull Quote
- 表格
- 圖片與 caption
- code block
- dropcap
- mobile typography

## 使用

上傳後可手動在文章頁 `<head>` 加入：

```html
<link rel="stylesheet" href="../assets/css/typography.css">
```

或執行：

```bash
python tools_article_typography_patch.py
```

自動替現有 `articles/*.html` 加入 typography CSS。
