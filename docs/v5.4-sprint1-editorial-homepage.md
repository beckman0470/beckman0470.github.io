# v5.4 Sprint 1｜Editorial Homepage

## 覆蓋

```text
engine/homepage.py
docs/v5.4-sprint1-editorial-homepage.md
```

## 功能

首頁升級為雜誌式 Editorial Homepage：

- Magazine Hero
- Cover Story
- Editor's Pick
- Today with Chicken Dad
- This Week
- Family Moments
- Research Notes
- Featured Series
- Brand Quote
- Magazine Footer

## 使用

覆蓋後執行：

```bash
python cms/build.py
```

即可重新產生 `index.html`。
