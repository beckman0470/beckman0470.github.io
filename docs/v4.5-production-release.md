# Chicken Dad Journal v4.5 Production Release

## 性質

這是 v4 系列第一個 Production Release，可直接整包覆蓋 GitHub Repository 根目錄。

## 整合內容

- v3.6 Stable Full Release

- v4.1 Related Stories + Previous / Next

- 文章頁上一篇 / 下一篇
- 時間軸頁
- 搜尋頁
- 標籤頁
- 訂閱頁
- SEO Meta
- JSON-LD Article Schema
