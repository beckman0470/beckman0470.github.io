## 儲存方式

GitHub Pages 沒有後端，所以不能直接寫回 Repository。

流程：

1. 開 `index.html?edit=1`
2. 點文字直接編輯
3. 按右下角「下載 index.html」
4. 將下載的 `index.html` 上傳覆蓋 GitHub 根目錄
5. Commit
