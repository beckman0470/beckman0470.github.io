# v4.4 SEO + Analytics

新增：

- engine/analytics.py
- engine/seo_plus.py

功能：
- Google Analytics 注入
- Article SEO Meta
- Keywords Meta
- Published Time Meta

整合方式：

```python
from engine.analytics import inject_analytics
from engine.seo_plus import inject_article_seo
```
在輸出 HTML 後依序呼叫：
```python
html = inject_article_seo(html, meta)
html = inject_analytics(html, measurement_id)
```
