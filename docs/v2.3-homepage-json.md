# v2.3 Homepage JSON Reader

## 覆蓋 / 新增

- index.html
- js/homepage.js

## 功能

首頁會嘗試讀取：

```text
data/stories.json
```

如果讀取失敗，會使用內建 fallback stories，所以首頁不會空白。

## 自動區塊

- 本月故事：抓 `hero: true` 或 `featured: true`
- 最近發生的故事：依日期排序最新三篇
- 繼續閱讀：依日期排序最新三篇
- 雞爸爸最近在研究：整理每篇 story 的 `research` 欄位
